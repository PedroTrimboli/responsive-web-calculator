# Responsive Web Calculator

A modern, responsive web calculator built with HTML, CSS and JavaScript.

## About the Project

This project is a responsive calculator with a modern interface and keyboard support.

It was developed to practice front-end development, JavaScript logic, responsive design and code organization.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Decimal numbers
- Keyboard support
- Responsive design
- Delete button
- Clear button
- Division by zero validation
- Invalid operation handling

## Technologies

- HTML5
- CSS3
- JavaScript

## Keyboard Controls

- Numbers: `0` to `9`
- Addition: `+`
- Subtraction: `-`
- Multiplication: `*` or `X`
- Division: `/` or `%`
- Calculate: `Enter`
- Delete: `Backspace`
- Clear: `Escape`
